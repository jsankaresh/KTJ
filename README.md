
# INotebook React

A notebook for storing text/notes into the server and retrieveing it remotely whenever required.This is MERN Stack based project.



## Features

- Remote access
- Cross Platform
- Simple UI
- Responsive,Dynamic
- Authentication
- AES Encryption
